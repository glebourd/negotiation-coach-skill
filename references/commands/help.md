# help — Command Reference

Display this when the user asks for help, seems unsure what to do next, or explicitly runs `help`.

## Output

```markdown
## Negotiation Coach — Command Reference

### Setup and Preparation
| Command | What it does |
|---|---|
| `kickoff` | Set up your profile, negotiation style, and coaching preferences. Start here. |
| `plan [negotiation]` | Build a full planning document — BATNA, RP, aspiration point, ZOPA, issue map, opening strategy. |
| `batna [negotiation]` | Deep BATNA analysis for your side and theirs. Rate strength, derive RP, and build an improvement plan if weak. |
| `score [negotiation]` | Build a point-value scoring system for multi-issue negotiations. Required before building MESOs. |
| `meso [negotiation]` | Build Multiple Equivalent Simultaneous Offers — reveal counterpart priorities without direct disclosure. |

### Landscape and Strategy
| Command | What it does |
|---|---|
| `situation` | Map the full negotiation landscape — parties, interests, BATNA, ZOPA, power dynamics, information gaps. |
| `leverage` | Identify and develop your leverage points across six categories. |
| `strategy` | Build the plan — anchor, concession ladder, scripts, red lines. |

### Practice and Execution
| Command | What it does |
|---|---|
| `simulate [format]` | Roleplay the negotiation conversation. Formats: offer, salary, raise, contract, pushback, silence, close, cold. |
| `counter [offer]` | Handle an incoming counteroffer in real time — analyze it, assess options, get exact language. |

### Analysis and Learning
| Command | What it does |
|---|---|
| `analyze` | Analyze a transcript, case study, or scenario — tactical breakdown, scoring, pattern match. |
| `debrief [negotiation]` | Post-negotiation capture — what happened, principle violations, lessons, identity update. |
| `reflect` | Extract generalizable principles from completed negotiations. Converts experience into identity. |
| `progress` | Review patterns, score trends, self-calibration, and your developing negotiation identity. |

### Navigation
| Command | What it does |
|---|---|
| `help` | Show this list. |

---

## Where to Start

**I have a specific negotiation coming up:**
→ `plan` or `situation` first, then `batna` if the BATNA is unclear, then `strategy`, then `simulate`

**I want to practice:**
→ `simulate offer` (or `salary`, `raise`, `contract`, `pushback`, `silence`)

**I just finished a negotiation:**
→ `debrief` — run this as soon as possible

**I want to see my progress:**
→ `progress`

**I'm not sure where to start:**
→ `kickoff`
```
